Agora que você já conhece o gênero bilhete, imagine que você irá deixar um recado para um familiar avisando a data de uma consulta médica marcada. Qual dos dois bilhetes é mais adequado para o contexto apresentado?
Arraste o bilhete escolhido para a porta da geladeira.

Prezada Maria.
Venho por meio deste informar que vossa senhoria tem consulta médica marcada na próxima quinta-feira. O honorável doutor receber-lhe-á no horário das 18h em seu consultório. É importante recordar de levar consigo seu documento de identificação.
Atenciosamente,
seu afetuoso familiar.

Apesar de ser um texto bem escrito, este não é o modelo ideal de um bilhete para um familiar. Como a ideia do bilhete é justamente a de informar brevemente sobre uma consulta médica, não é necessário o uso de linguagem formal e da norma padrão da língua portuguesa. Uma linguagem simples e direta é o ideal para esta situação comunicativa.

Maria, marquei médico para você.
A consulta vai ser na próxima quinta-feira, às 18h.
Não esqueça o documento, viu?
Um abraço!

Parabéns! Este é o modelo ideal de um bilhete para um familiar. Note que ele apresenta uma linguagem informal, que é mais simples e objetiva, com o intuito de informar sobre uma consulta médica.
